## Technologies

- Vue (+Router)
- Typescript
- Tailwind
  
 
## Start Setup
With NPM:

```bash
$ npm install
```

With Yarn:

```bash
$ yarn add
```

Render Tailwind:

```
npm/yarn run tailwind
```
